# mariowOS
## How Windows should have been.

Welcome to mariowOS, a progect made to revive Windows by debloating, improving the user interface and overall the system stability.

This is the **only** official repo, and it is under "mariowstech/mariowOS". Drive around a little bit to get you familiar with this project.
